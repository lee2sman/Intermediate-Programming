## Procedure

1. Create a Voice User Interface (VUI)
2. Create a Lambda AWS Function
3. Connect the 2 together
4. Test it
5. Modify the code

## Terminology
* *activation word* = app name
* *utterances* = function calls
* *intents* = functions
* *slots* = variables
